# NSE Market Analysis using Python

This project collects and analyzes data on companies listed in the NIFTY 50 index.

## Features

- Web scraping using BeautifulSoup
- Data cleaning and analysis using Pandas
- Sector-level exploratory data analysis
- Data visualization using Matplotlib

## Technologies

Python  
Pandas  
NumPy  
Matplotlib  
BeautifulSoup  

## Project Structure

nse-market-analysis
│
├── scraper.py
├── analysis.py
├── data/
├── visuals/
└── README.md

## Example Insight

Preliminary analysis shows sector concentration among Financial Services and IT companies in India's large-cap stock index.
